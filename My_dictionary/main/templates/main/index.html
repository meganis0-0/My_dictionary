My Dictionary

Это веб-приложение реализует работу словаря. Оно хранит в базе данных слово и его перевод. На странице со списком всех слов есть возможность поиска по слову и его переводу. При нажатии на кнопку "Добавить" открывается страница добавления нового слова и его перевода. При нажатии на кнопку "Удалить" слово удаляется из базы данных.
